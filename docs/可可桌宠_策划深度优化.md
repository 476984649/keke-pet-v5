## 一、2025-2026 桌宠新趋势洞察

v2.0 对标的是 2020-2024 的产品。2025-2026 出现了几个新方向，可可必须跟上：

### 1.1 AI 对话桌宠（最大变量）

| 产品 | 形态 | 用户量 |
|------|------|--------|
| **Digi AI** | AI虚拟女友/男友，桌面+手机双端 | 百万级 |
| **Character.AI桌面版** | AI角色对话，含猫娘等宠物角色 | 千万级 |
| **NVIDIA ACE桌宠demo** | 本地AI推理，实时语音对话 | 实验阶段 |

> **趋势**：用户不再满足于"只会做动画的桌宠"，期待角色能"理解自己"。可可应该是**第一个能记住你的AI桌宠猫**。

### 1.2 桌面小组件化

| 产品 | 特征 |
|------|------|
| **Rainmeter** | 桌面小组件之王，千万用户 |
| **Widget Launcher** | Win11 小组件，2025年爆发 |
| **Lively Wallpaper** | 动态壁纸+互动，Steam 好评如潮 |

> **趋势**：桌宠不只是"宠物"，而是"桌面生态的一部分"。可可应该和桌面深度融合。

### 1.3 直播/社交联动

| 场景 | 说明 |
|------|------|
| Twitch直播互动 | 观众打赏触发桌宠反应 |
| Discord Rich Presence | 桌宠状态显示在Discord |
| TikTok/Reels短视频 | 可可名场面天然适合短视频传播 |

### 1.4 本地AI能力成熟

2025-2026 年，**本地小模型已可运行在消费级PC上**：
- Ollama + 小模型（1-3GB）可本地运行
- WebLLM 可在浏览器内运行
- ONNX Runtime 可嵌入 Tauri 应用

> 可可可以拥有一个"本地AI大脑"，不需要联网。

## 二、可可"灵魂"深化方案

### 2.1 🧠 可可的 AI 人格引擎（核心建议）

```
┌─────────────────────────────────────────────┐
│              可可 AI 人格引擎                  │
│                                               │
│  ┌─────────┐  ┌─────────┐  ┌─────────────┐  │
│  │ 情绪系统 │  │ 记忆系统 │  │ 对话系统     │  │
│  │         │  │         │  │             │  │
```

#### 情绪系统（新增）

可可不再永远开心。她有自己的小情绪：

| 情绪 | 触发 | 表现 | 持续 |
|------|------|------|------|
| 😊 开心 | 被摸头、喂食 | 尾巴翘高、主动蹭鼠标 | 30min |
| 😴 困倦 | 下午2-4点、深夜 | 打哈欠频繁、走路变慢 | 自然消退 |
| 😤 小脾气 | 被拎太多次、被双击惊吓 | 背对你、尾巴甩来甩去 | 10min |
| 🥺 想你 | 电脑长时间锁屏后回来 | 扑上来蹭屏幕、疯狂喵叫 | 5min |
| 😰 害怕 | 打雷音效、突然大音量 | 躲到屏幕角落发抖 | 直到安抚 |
| 🎉 兴奋 | 猫薄荷、好感度里程碑 | 满屏乱跑、打滚 | 5min |

#### 记忆系统（新增）

> 这是可可区别于所有竞品的核心——**她记得你**。

| 记忆类型 | 示例 | 用途 |
|------|------|------|
| 你的名字 | "宣儒今天来得好早呀" | 对话个性化 |
| 作息习惯 | 知道你通常9点开电脑 | 准时在桌面等你 |
| 互动偏好 | 你喜欢摸头 > 喂食 | 主动求摸头 |
| 特殊日子 | 生日、可可"来你家"纪念日 | 特殊庆祝动画 |
| 最近事件 | "昨天你摸了我47次！" | 对话素材 |

#### 对话系统（新增）

```
用户："可可今天乖不乖？"
可可：气泡框 "喵~（当然乖啦，就踩了一次键盘）"

用户："我压力好大"
可可：跳上屏幕中央，蹭屏幕 + 呼噜声

用户：什么都不说，只是鼠标在可可附近晃
可可：用爪子碰碰光标，歪头 "喵？"
```

> 不需要复杂NLP，用**模板+情绪+记忆**就能产生"可可懂我"的错觉。

### 2.2 📔 可可的日记（新增）

每天关机前，可可生成一篇"日记"：

```

今天宣儒工作了6小时，摸鱼2次（被我拍屏幕警告了！）
被摸头23次，喂了2条小鱼干。
下午3点我睡着了，宣儒没有叫醒我，他真好。
娜娜今天没来串门，有点想她。

今日心情：😊
好感度：72 → 85 (+13)
```

> **传播钩子**：用户会截图可可的日记分享——"看我家猫写的日记！"

可可有自己的小任务，完成时用户获得奖励：

| 任务 | 描述 | 奖励 |

| 🪲 抓桌面虫 | 随机出现小虫子，可可去抓 | 好感+3 |

## 三、传播钩子深化方案

### 3.1 🎬 一键 GIF 生成器（核心建议）

> 桌宠最自然的传播形式是 GIF，但用户不会录屏→剪辑→导出。

**可可内置 GIF 录制**：
- 右键 → "录制名场面"
- 自动录制接下来 5 秒
- 一键复制到剪贴板 / 保存到桌面
- 自动加上可可水印 + #可可桌宠 标签

**预设录制模板**：
| 模板 | 触发 | GIF内容 |
|------|------|---------|
| "我家猫疯了" | 猫薄荷后自动提示 | 可可满屏乱跑 |
| "被抓包了" | 摸鱼警告时 | 可可拍屏幕 |
| "猫的日常" | 随机 | 可可舔毛→打哈欠→睡觉 |
| "贴贴" | 连续摸头10次 | 可可蹭鼠标 |

### 3.2 🎭 可可表情包工坊（新增）

可可的动画帧天然适合做表情包：

```
右键 → 表情包工坊 → 选择模板
                           │
              ┌────────────┼────────────┐
              ↓            ↓            ↓
         "不想上班"    "摸鱼中"     "贴贴"
         （可可趴着）  （可可偷看） （可可蹭）
                           │
                    一键导出PNG/GIF
                    自动加文字+可可水印
```

### 3.3 🎵 音乐响应模式（新增）

> 2025年 TikTok 最火的桌宠视频：猫随音乐跳舞。

可可检测到系统播放音频时：
- 节奏感强的音乐 → 可可点头/摇尾巴踩节拍
- 舒缓音乐 → 可可趴下睡觉
- 用户可切换"打碟可可"模式：可可戴墨镜打碟

### 3.4 🎄 实时节日彩蛋（新增）

| 节日 | 可可行为 | 传播力 |
|------|---------|:--:|
| 🧧 春节 | 穿红色小马甲，叼春联 | ⭐⭐⭐⭐⭐ |
| 🎃 万圣节 | 戴小南瓜帽，背景变暗 | ⭐⭐⭐⭐ |
| 🎄 圣诞节 | 戴圣诞帽，桌面飘雪花 | ⭐⭐⭐⭐ |
| 🎂 用户生日 | 可可叼蛋糕+气球动画 | ⭐⭐⭐⭐⭐ |
| 📅 可可在你电脑的"周年日" | 特殊庆祝 | ⭐⭐⭐⭐ |

### 3.5 🔗 社交平台深度集成（新增）

## 五、商业化深化方案

### 5.1 三层付费模型

```
免费层（80%用户）
├── 可可基础版
├── 全部行为+情绪+记忆
├── 5种基础音效
└── 每日3条日记

会员层 $2.99/月（15%用户）
├── 无限日记+日记导出
├── AI对话（本地小模型）
├── 可可小组件
├── 表情包工坊
├── GIF生成器无水印
├── 节日限定装扮自动推送
└── Discord Rich Presence

买断层 $9.99（5%用户）
├── 永久会员全部功能
├── 娜娜+阿里角色包
├── 全部装扮
└── 早期支持者专属徽章
```

### 5.2 Steam 定价策略

| 版本 | 价格 | 内容 |
|------|------|------|
| 免费版 | $0 | 可可基础（含广告位→装扮推荐） |

## 六、技术实现路径

### 6.1 AI 人格引擎实现

```
v0.1：规则引擎（情绪用 if/else，对话用模板匹配）
       ↓

       ↓
```

```
本地预置 5 种基础音效（MP3）
     +
社区音效包（Steam工坊）

```

