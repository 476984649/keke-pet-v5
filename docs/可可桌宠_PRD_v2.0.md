# 🐱 可可桌宠（Keke Desktop Pet）产品需求文档 v2.0

> 版本：v2.0 | 日期：2026-07-07 | 基于 v1.0 + 竞品深度调研优化

---

## 一、市场调研

### 1.1 竞品深度拆解

| 产品 | 类型 | 成功秘诀 | 对可可的启示 |
|------|------|---------|-------------|
| **Desktop Goose** | 恶搞型 | 恶搞=传播素材，用户主动录屏分享 | 可可需要"名场面" |
| **Shimeji** | 陪伴型 | 开源+社区角色包，活了15年 | v1.0就做UGC图片替换 |
| **Bongo Cat** | 互动型 | 50%魅力来自音效，极简即传播 | 音效必须在MVP |
| **eSheep** | 怀旧型 | 情怀，20年未更新 | — |
| **Desktop Mate** | 3D型 | 3D空间感+VR手势 | 远期方向 |
| **Virtual Cottage** | 氛围型 | 小众陪伴 | — |
| **Nekopara桌宠** | 粉丝向 | Live2D呼吸感+语音，IP驱动 | 正弦波浮动零成本实现 |

### 1.2 市场空白

```
        互动性高
           │
  Desktop Goose ●    │    ● Nekopara
           │
           │     ★ 可可桌宠定位
           │    （治愈陪伴 + 趣味互动 + 养成）
           │
  ──────────┼──────────→ 养成深度
           │
  Shimeji ● │    ● Virtual Cottage
           │
    Bongo Cat ●
           │
        互动性低
```

**核心洞察**：市面上缺少一款**既有深度养成、又有趣味互动、画风治愈**的桌宠。Desktop Goose 好玩但没养成，Shimeji 有生态但没成长感。

### 1.3 对标策略

- **行为系统** 对标 Shimeji（走动、爬窗、多实例）
- **互动趣味** 对标 Desktop Goose（惊喜事件、温柔捣乱）
- **动画质感** 对标 Bongo Cat（帧动画流畅可爱）
- **养成深度** 自研（好感度、装扮、成就）
- **猫设驱动** 独家（可可"认真蓝山双布偶"性格贯穿所有功能）

### 1.4 v1.0 三大缺陷（v2.0 已修复）

1. **缺猫设驱动** — 功能是通用猫咪，不是"可可"
2. **缺传播钩子** — 用户不因"猫对我好"而分享，因"猫干了搞笑的事"
3. **MVP太臃肿** — 2周5模块不现实

## 二、产品定位

### 2.1 一句话介绍

> 一只叫可可的蓝山双布偶猫住在你的电脑里，陪你工作、温柔捣乱、慢慢长大。

### 2.2 目标用户

| 画像 | 特征 | 需求 |
|------|------|------|
| 学生/白领 | 长时间面对电脑 | 陪伴减压、摸鱼乐趣 |
| 猫奴 | 喜欢猫但无法养 | 云养猫替代满足 |
| 二次元用户 | 喜欢可爱角色 | 收集装扮、截图分享 |
| Steam玩家 | 轻度游戏习惯 | 后台挂机、成就驱动 |

### 2.3 可可猫设

| 属性 | 设定 |
|------|------|
| 品种 | 蓝山双布偶猫（Blue Bicolor Ragdoll） |
| 外貌 | 圆脸、蓝眼、面部倒V白色花纹、白四肢、蓝灰耳朵尾巴 |
| 性格 | 认真、温柔、偶尔犯困、喜欢被摸头 |
| 口头禅 | "喵~"（不同语调表达不同情绪） |

## 三、猫设驱动设计（v2.0 新增）

> 功能不再通用，每个行为都体现"认真布偶猫可可"的性格

### 3.1 可可专属行为

| 行为 | 触发条件 | 猫设体现 | 传播价值 |
|------|---------|---------|:--:|
| 📋 **整理桌面** | 桌面图标散乱 | 可可把图标推到网格对齐 | ⭐⭐⭐ |
| 👀 **监工模式** | 检测到IDE/Office全屏 | 安静趴在任务栏，偶尔抬头看你 | ⭐⭐ |
| ⚠️ **摸鱼警告** | 连续30min在浏览器/游戏 | 跳上屏幕中央拍屏幕 | ⭐⭐⭐⭐⭐ |
| 🎁 **叼礼物** | 好感度里程碑 | 叼小花/叶子放在鼠标旁 | ⭐⭐⭐ |
| 😴 **陪加班** | 超过晚10点还在用电脑 | 困得东倒西歪但坚持不睡 | ⭐⭐⭐⭐ |

### 3.2 传播钩子：可可的"名场面"

> 模仿 Desktop Goose 但改为温柔捣乱——低频+无害=可爱

| 事件 | 频率 | 描述 |
|------|------|------|
| ⌨️ **踩键盘** | 每周1-2次 | 在键盘区域走过，随机打出字符串 |
| 🖥️ **挡屏幕** | 每月1-2次 | 突然趴在屏幕中央，3秒后走开 |
| 📱 **推图标** | 随机 | 把桌面一个图标推歪一点点 |
| 🖱️ **偷鼠标** | 极低概率 | 追光标→抓住→拖着走一段 |

## 四、功能需求

### 4.1 桌面存在感

- 透明窗口常驻桌面最上层
- 窗口约 150×200px
- 拖拽移动可可位置
- 启动时从屏幕边缘走进来
- **正弦波呼吸浮动**：待机时 ±3px 上下浮动 + ±2% 缩放（4秒周期）

### 4.2 自主行为系统（状态机）

```
         ┌──────────┐
    ┌───→│  睡觉 💤  │←───┐
    │    └─────┬─────┘    │
    │          │ 醒了      │ 5分钟无操作
    │          ↓          │
    │    ┌──────────┐    │
    │    │  待机 😺  │←───┘
    │    └─────┬─────┘
    │          │
    │     ┌────┼────┐
    │     ↓    ↓     ↓
    │  🚶走路  🪟爬窗  🎯盯鼠标
    │     │    │     │
    │     └────┼─────┘
    │          ↓
    └──── 回到待机
```

| 行为 | 触发 | 持续 | 动画 |
|------|------|------|------|
| 待机 | 默认 | 5-15秒 | 眨眼、摇尾巴、呼吸浮动 |
| 走路 | 随机 | 3-8秒 | A→B位移 |
| 爬窗口 | 靠近窗口边缘 | 直到掉落 | 攀爬、悬挂、掉落 |
| 盯鼠标 | 鼠标靠近 | 持续 | 头随光标转 |
| 睡觉 | 5min无操作 | 直到唤醒 | 蜷缩、呼吸起伏 |
| 打哈欠 | 随机 | 2秒 | 张嘴、伸懒腰 |
| 舔毛 | 待机中随机 | 3秒 | 舔爪、洗脸 |

### 4.3 鼠标互动

| 操作 | 行为 | 反馈 |
|------|------|------|
| 单击 | 摸头 | 眯眼喵~ +1好感 |
| 双击 | 弹跳惊吓 | 跳起落地瞪你 |
| 拖拽 | 拎起 | 四肢垂挂委屈 |
| 右键 | 菜单 | 喂食/装扮/状态/退出 |
| 鼠标晃动 | 伸爪抓光标 | 爪子跟随轨迹 |

### 4.4 好感度与养成

```
好感度 0──100──300──600──1000
       │    │     │     │
       陌生  熟悉  亲密  家人
       │    │     │     │
       基础   解锁   解锁   解锁
       行为   新动画  装扮   彩蛋
```

- 摸头 +1、喂食 +5、在线每小时 +10
- 好感度解锁新行为和互动
- 每天不互动 -2

### 4.5 喂食系统

| 食物 | 效果 | 冷却 |
|------|------|------|
| 🐟 小鱼干 | 好感+5，兴奋10秒 | 5分钟 |
| 🥫 猫罐头 | 好感+15，满足睡觉 | 30分钟 |
| 🌿 猫薄荷 | 疯狂状态 | 1小时 |

### 4.6 增值功能（v1.0+）

- **装扮系统**：帽子、围巾、眼镜，节日限定
- **多猫模式**：可可+娜娜+阿里同屏，猫咪互动
- **桌面工具**：番茄钟、喝水提醒、整点报时
- **成就系统**：初次见面、猫奴认证、小鱼干大亨、熬夜冠军
- **Steam 创意工坊**：用户替换角色图片、分享角色包

## 七、技术方案（v2.0 调整）

### 7.1 技术选型

| 层 | 方案 | 理由 |
|----|------|------|
| 框架 | **Tauri** | 包体<10MB，内存<30MB，原生透明窗口 |
| 前端 | HTML5 Canvas + CSS | 帧动画流畅 |
| 动画 | CSS Sprite + 正弦波位移 | 呼吸感零成本 |
| 音效 | Web Audio API | 低延迟播放 |
| 存储 | SQLite（本地） | 好感度、设置 |
| 打包 | tauri-bundler | 一键 exe/msi |

> v1.0 选 Electron → v2.0 改 Tauri：包体从 80MB 降到 5MB，内存从 100MB 降到 20MB。

### 7.2 动画资源

| 动画 | 帧数 | 尺寸 | 格式 |
|------|------|------|------|
| 待机眨眼 | 8帧 | 150×200 | PNG序列 |
| 走路 | 12帧 | 200×200 | PNG序列 |
| 爬窗 | 10帧 | 200×300 | PNG序列 |
| 睡觉 | 6帧循环 | 200×150 | PNG序列 |
| 摸头反应 | 6帧 | 150×200 | PNG序列 |
| 吃鱼干 | 8帧 | 200×200 | PNG序列 |
