## 一、技术选型论证

### 1.1 框架横向对比

| 框架 | 包体 | 内存(空闲) | 内存(满负载) | 启动速度 | 透明窗口 | 跨平台 | 社区活跃度 | 综合评分 |
|------|:---:|:---------:|:-----------:|:-------:|:-------:|:-----:|:---------:|:-------:|
| **Electron** | 80-120MB | 80-120MB | 180MB+ | 3-5s | ✅ | ✅ | ⭐⭐⭐⭐ | ❌ 太重 |
| **Tauri v2** | **3-8MB** | **15-25MB** | **50-80MB** | **0.8-1.2s** | ✅ | ✅ | ⭐⭐⭐⭐⭐ | **✅ 首选** |
| **Unity URP** | 30-60MB | 60-100MB | 150MB+ | 2-4s | ⚠️ 复杂 | ✅ | ⭐⭐⭐⭐ | ⚠️ 过度 |
| **Godot 4** | 20-40MB | 40-70MB | 100MB+ | 1.5-3s | ✅ | ✅ | ⭐⭐⭐⭐ | ✅ 备选 |
| **Flutter** | 10-20MB | 30-50MB | 80MB+ | 1-2s | ⚠️ 实验性 | ✅ | ⭐⭐⭐ | ❌ 不稳定 |
| **Sciter** | 1-3MB | 10-20MB | 40MB+ | 0.5s | ✅ | ⚠️ Win为主 | ⭐⭐ | ⚠️ 小众 |

### 1.2 最终选型：Tauri v2

**核心理由**：

```
裸 Tauri + Webview2
│
├── 启动速度 0.8s （vs Electron 4s）
├── 内存占用 25MB 空闲 （vs Electron 100MB）
├── 包体 5MB 打底 （vs Electron 80MB）
├── 透明窗口 原生支持（Win + macOS + Linux）
├── IPC 零开销（Rust ←→ Webview 直接调用）
└── Web 技术栈（前端开发者友好）
```

> 桌宠是**常驻后台**应用，用户开机就启动，内存/CPU/启动速度是第一位体验。Electron 不适合这个场景。

#### 状态转移核心逻辑

```rust
impl KekeState {
    /// 状态转移主方法 — 根据当前状态和外部输入决定下一状态
    pub fn transition(&self, input: &InputEvent, ctx: &Context) -> KekeState {
        match (self, input) {
            // 任何人机互动 → 打断当前状态
            (_, InputEvent::ClickedAt(_)) => KekeState::Petting,
            (_, InputEvent::DragStart(_)) => KekeState::Held,

            // Idle → 随机触发
            (KekeState::Idle, InputEvent::Timeout) => {
                match ctx.random_behavior() {
                    Behavior::Walk => KekeState::Walking(Direction::Random),
                    Behavior::Climb => KekeState::Climbing(ctx.nearest_window()),
                    Behavior::Watch => KekeState::WatchingMouse,
                    Behavior::Organize => KekeState::OrganizingDesk,
                    Behavior::Gift => KekeState::GiftDelivery,
                    _ => KekeState::Idle,
                }
            },

            // 5分钟无操作 → 睡觉
            (_, InputEvent::LongIdle(300)) => KekeState::Sleeping,

            // 可可专属行为
            (KekeState::Idle, InputEvent::FullscreenAppDetected) => KekeState::SupervisorMode,
            (KekeState::SupervisorMode, InputEvent::LongBrowserActivity(1800)) => KekeState::SlackerWarning,
            (KekeState::SlackerWarning, InputEvent::Timeout) => KekeState::SupervisorMode,

            // 喂食
            (KekeState::Idle, InputEvent::Feed(food)) => KekeState::Eating(food),

            // 默认保持当前状态
            _ => self.clone(),
        }
    }
}
```

```typescript
// 核心动画加载器
class KekeSpriteLoader {
    private textures: Map<string, Texture[]> = new Map();

    async loadAnimation(name: string, frameCount: number): Promise<void> {
        const frames: Texture[] = [];
        for (let i = 1; i <= frameCount; i++) {
            const path = `assets/keke_${name}_f${String(i).padStart(2, '0')}.png`;
            frames.push(await Texture.from(path));
        }
        this.textures.set(name, frames);
    }

    createAnimatedSprite(name: string): AnimatedSprite {
        const frames = this.textures.get(name);
        if (!frames) throw new Error(`Animation not loaded: ${name}`);
        const sprite = new AnimatedSprite(frames);
        sprite.anchor.set(0.5, 0.5);
        return sprite;
    }
}
```

### 3.4 情绪引擎

```typescript
interface MoodState {
    current: Mood;
    startTime: number;
    duration: number; // 毫秒
}

type Mood = 'happy' | 'sleepy' | 'grumpy' | 'lonely' | 'scared' | 'excited';

class MoodEngine {
    private mood: MoodState = { current: 'happy', startTime: Date.now(), duration: 30 * 60_000 };

    update(context: Context): void {
        if (this.isExpired()) {
            this.mood = this.computeNextMood(context);
        }
    }

    private computeNextMood(ctx: Context): MoodState {
        // 好感度低 → 更容易 grumpy
        // 深夜 → sleepy
        // 刚互动完 → happy
        // 长时间未互动 → lonely
        // 猫薄荷中 → excited
        // ...
        return { current: 'happy', startTime: Date.now(), duration: 30_000 };
    }

    get bubbleText(): string {
        const texts: Record<Mood, string[]> = {
            happy:  ['喵~今天好开心！', '贴贴~', '摸摸头！'],
            sleepy: ['好困哦...zzz', '想睡觉了...', '哈欠~'],
            grumpy: ['哼！', '不理你了', '尾巴甩甩'],
            lonely: ['你去哪了...', '好想你呀', '喵...'],
            scared: ['有声音！', '害怕...', '抖抖'],
            excited: ['小鱼干！！', '好耶好耶！', '跑跑跳跳～'],
        };
        const pool = texts[this.mood.current];
        return pool[Math.floor(Math.random() * pool.length)];
    }
}
```

```rust
pub struct AIRequest {
    pub intent: AIIntent,
    pub user_input: Option<String>,
    pub context: AIContext,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub enum AIIntent {
    /// 对话：用户对可可说话
    Chat,
    /// 生成日记：根据今日互动生成日记内容
    GenerateJournal,
    /// 情绪建议：根据当前情况推荐可可行文
    BehaviorSuggestion,
    /// 取名：给新装扮/事物取名
    NameGeneration,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct AIContext {
    pub current_mood: String,
    pub affection_level: u32,
    pub time_of_day: String,
    pub recent_events: Vec<String>,
    pub user_name: Option<String>,
    pub user_habits: Vec<String>,
}

/// AI 响应
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct AIResponse {
    pub text: Option<String>,
    pub suggested_mood: Option<String>,
    pub suggested_behavior: Option<String>,
    pub data: Option<serde_json::Value>,
}

/// AI 提供者 Trait
#[async_trait]
pub trait AIProvider: Send + Sync {
    /// 提供者名称
    fn name(&self) -> &'static str;

    /// 处理 AI 请求
    async fn process(&self, request: AIRequest) -> Result<AIResponse, String>;
}
```

### 5.4 规则引擎 Fallback（默认启用）

```rust
/// 默认规则引擎 — 无需任何 AI 模型即可运行
pub struct RuleEngine;

impl RuleEngine {
    pub fn generate_response(&self, context: &Context, intent: AIIntent) -> AIResponse {
        match intent {
            AIIntent::Chat => AIResponse {
                text: Some(self.pick_template(&context)),
                ..Default::default()
            },
            AIIntent::GenerateJournal => AIResponse {
                text: Some(self.generate_diary(context)),
                ..Default::default()
            },
            _ => AIResponse::default(),
        }
    }

    fn pick_template(&self, ctx: &Context) -> &'static str {
        match (ctx.mood, ctx.idle_seconds) {
            (_, s) if s < 10 => "喵~",
            (Mood::Happy, _) => "今天好开心！",
            (Mood::Sleepy, _) => "好困哦 zzz",
            (Mood::Lonely, _) => "你去哪了...",
            _ => "喵?",
        }
    }
}
```
